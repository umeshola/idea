### IdeaShare!
*Quickstart server*
```
clone
cd server
npm install or yarn add
nodemon or node index.js
```

*Quickstart client*
```
cd client
npm install
npm run dev
```

This is an idea-sharing platform where you can only share your ideas and get funding, workers, and friends.
